### Jupyter notebooky: 

1970_90_preprocessing.ipynb - čistenie pôvodných súborov, lematizácia názvov a aplikovanie tfidf 

2007_22_preprocessing.ipynb - čistenie pôvodných súborov, lematizácia názvov a aplikovanie tfidf
